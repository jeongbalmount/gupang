# 🚀Gupang

## 개발 동기 및 앱 설명

  쇼핑몰api 구현 프로젝트
  
## 사용한 기술 및 클라우드

  언어: `JAVA`
  
  프레임워크: `Spring`
  
  데이터베이스 및 orm: `mysql`, `JPA`, `Spring data JPA`, `Quarydsl`
  
  캐싱 및 세션 관리: `redis`
  
  클라우드: `AWS`, `ec2`, `elb`, `rds`, `elasticache`
  
  API: `Swagger`
  
  Test: `JUnit5`
